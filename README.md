# C212-SP26-Final-JavaChess
## Instructions:
- Download and open in Intellij IDEA
- Run Main.java
- See instructions.txt for more info

## Features:
- Includes chess pieces with move and capture rules
- Checkmate, resign, and draw game states
- Shows possible moves when a piece is selected
- Prevents illegal moves from being played
- Includes basic pawn promotion (only can promote to Queen)
- Includes GUI to play the game on
- Writes game results to file
